## Development Transparency

### **Open Source Commitment**
All DIG Network code is open source and available on GitHub:

- **[DIG CLI](https://github.com/DIG-Network/dig-chia-cli)** - Command-line interface for developers
- **[DIG SDK](https://github.com/DIG-Network/dig-chia-sdk)** - Developer toolkit and libraries  
- **[DIG Node](https://github.com/DIG-Network/chia-dig-node)** - Decentralized storage node software
- **[Content Server](https://github.com/DIG-Network/dig-content-server)** - Content delivery infrastructure
- **[Propagation Server](https://github.com/DIG-Network/dig-propagation-server)** - Network propagation system
