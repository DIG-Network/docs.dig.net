# A Proof-of-Stake Layer 2 on Chia

*Part 1 · Consensus*

The foundation: validators register by locking XCH on Chia L1, finality is a Groth16 proof plus a
BLS aggregate signature posted each epoch, and L2 state commits to Chia L1 through five CLVM
puzzles. Covers consensus, L1 anchoring, the validator lifecycle, and binaries.
